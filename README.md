Epigraph
======

About
---
Working prototype of a real-time collaborative workspace and task management platform.

Technologies
---
* TypeScript
* React 18
* Next.js 14 
* Clerk 
* TailwindCSS
* Radix UI
* Shadcn/ui
* LiveBlocks
* Convex BaaS
* Unsplash API

Demo URL
---
